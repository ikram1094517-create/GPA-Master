# GPA Master (Academic Calculator)
Technologies: Developed using HTML5, CSS3 (Custom Properties), Vanilla JavaScript, and libraries (html2canvas, jsPDF).
Description: A specialized GPA management tool for Algerian universities featuring dynamic major selection, automatic optimizedy detection via Geolocation, and high-quality report exporting in PDF/Image formats. It includes a responsive multilingual interface with dark mode support for an optimized user experience.
